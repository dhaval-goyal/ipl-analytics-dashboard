# ipl-analytics-dashboard
IPL Analytics Dashboard built with Python, Pandas, Plotly and Streamlit

## Dashboard Questions
1. Which team won most matches per season?
2. Who are the Top 10 batsmen by total runs?
3. What is the toss win % by venue?
4. Who are the best bowlers by economy (min 20 overs)?
5. How have total runs per season changed over time?
